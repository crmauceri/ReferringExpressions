## Prerequisites

-pytorch 0.4


## Install

- Clone recursive 
```
git clone --recurse-submodules https://github.com/crmauceri/ReferExpGeneration.git
```
- Make and install mask extention
```
cd pyutils/refer_python3
make
pip install -e .
```
- Install pyutils 
```
cd ../..
pip install -e pyutils/
```
